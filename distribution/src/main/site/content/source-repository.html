Overview

JBehave uses Git to manage its source code.

Web Access

Git allows web access:

http://github.com/jbehave/jbehave-web

Anonymous Access

git clone git://github.com/jbehave/jbehave-web.git

Committer Access

git clone ssh://git@github.com/jbehave/jbehave-web.git

Access through a proxy

Git support HTTP protocol for cloning, which normally is allowed in corporate firewalls

git clone https://github.com/jbehave/jbehave-web.git

You may need to set http.proxy via git-config.
