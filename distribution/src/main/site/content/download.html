Stable releases

Stable releases can be found on the Sonatype Nexus Repository Manager (which is synched with Maven Central Repository).

Latest Snapshots

The snapshot builds of JBehave can be found on the Sonatype Nexus Repository Manager. Be sure to choose the snapshot with the latest version and the filename with the latest timestamp.
